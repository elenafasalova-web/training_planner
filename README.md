# Training Planner

**Автор:** Елена Фасалова

## Описание
Это простое GUI-приложение для планирования тренировок. Вы можете добавлять тренировки, фильтровать их по типу и дате, сохранять данные в JSON.

## Использование
1. Запустите "main.py".
2. Введите дату (ГГГГ-ММ-ДД), тип тренировки и длительность.
3. Нажмите "Добавить тренировку".
4. Для фильтрации введите нужные параметры и нажмите соответствующие кнопки.
5. Все данные сохраняются автоматически в "data.json".

## Требования
- Python 3.x
- Tkinter
- Git для контроля версий
